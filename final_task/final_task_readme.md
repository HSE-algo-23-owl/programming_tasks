#Постановка задачи
Необходимо реализовать алгоритм Нидлмана-Вунша.
Алгоритм Ниддлмана-Вунша используется для нахождения оптимального выравнивания двух последовательностей путем минимизации штрафов за пропуски и несовпадения символов. Сначала создается матрица и заполняется значениями, соответствующими штрафам выравнивания. Затем определяется обратный путь с наименьшим штрафом.
Входные данные: 2 последовательности (строки), значение штрафов за совпадение, несовпадение и пропуск.
Выходные данные: 2 последовательности (строки).
Чаще всего алгоритм Нидлмана-Вунша используется в биоинформатике при построении выравнивай аминокислотных и нуклеотидных последовательностей.
